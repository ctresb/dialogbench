Como Usar o Dialog Bench

touch_app

Navegação

Clique e arraste no fundo para mover a área de trabalho. Use a roda do mouse ou os botões de zoom para aumentar ou diminuir o zoom.

add_box

Criar Blocos

Clique no botão "Novo Bloco" ou pressione Shift + N para criar um novo bloco de diálogo. Clique e arraste o cabeçalho do bloco para movê-lo.

edit

Editar Diálogos

Clique nas linhas de texto para editá-las. Use Shift + Enter para adicionar uma nova linha. Pressione Backspace em uma linha vazia para deletá-la.

question_answer

Adicionar Respostas

Clique no botão + na seção de respostas para adicionar opções de escolha. Conecte respostas a outros blocos digitando o ID do bloco de destino.

settings

Adicionar Variáveis

Clique no botão + na seção de variáveis para adicionar valores customizados. Use para armazenar estados do jogo, flags ou metadados.

link

Conexões

As conexões entre blocos são desenhadas automaticamente quando você vincula uma resposta a um bloco. As linhas mostram o fluxo de conversação.

save

Salvar e Carregar

Use "Salvar JSON" para exportar seu trabalho. "Carregar JSON" permite importar projetos salvos. Seu trabalho é automaticamente salvo localmente.

grid_4x4

Grid Snapping

Ative o botão de grid para alinhar blocos automaticamente em uma grade. Útil para manter sua árvore de diálogos organizada.

lightbulb

Dicas

  • Use cores diferentes para categorizar tipos de respostas (hostil, amigável, neutro).
  • Mantenha IDs de blocos organizados para facilitar a conexão entre diálogos.
  • Copie textos usando os botões de copiar ao lado de cada elemento.
  • Use variáveis para marcar pontos importantes na narrativa.
